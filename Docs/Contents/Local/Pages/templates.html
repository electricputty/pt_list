Template Tags and Parameters

Primary Tag Pair

Calling your P&T List field with a tag pair allows you to fully customize the output:

<p>I like {favorite_things backspace="2"}{item}, {/favorite_things}.</p>

Single Variable Tags

The following single variable is available within your tag pair:

{item}
The current item

:ol and :ul Tags

Returns an ordered/unordered list of each of your selected options

<h3>My favorite things:</h3>
{favorite_things:ul}
